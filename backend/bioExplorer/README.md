# `bioExplorer` App

The `bioExplorer` app is a **placeholder** for a planned exploration/discovery feature. It is currently stubbed out with empty models and views.

---

## Current State

| File          | Status                        |
| ------------- | ----------------------------- |
| `models.py`   | Empty — no models defined yet |
| `views.py`    | Empty — no views defined yet  |
| `migrations/` | Initial migration only        |

---

## Planned Purpose

BioExplorer is intended to expose a free-roam mode where users can explore the global animal sighting map without a time-limited game round. Potential features include:

- Browsing animals by region / biome
- Viewing heatmaps of sighting density
- Species comparison and filtering

---

## Dependencies

- Will depend on **`animal`** for all sighting and species data
